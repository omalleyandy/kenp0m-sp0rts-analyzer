## What is FanMatch?

FanMatch is KenPom's game prediction system that provides daily predictions for NCAA Division I Men's Basketball games. It uses KenPom's efficiency ratings to generate:

1. **Predicted Scores** for both teams
2. **Win Probability** (0-100%) for the home team
3. **Predicted Tempo** (total possessions)
4. **Thrill Score** (expected game excitement, 0-100)

### Field Definitions

| Field | Description |
|-------|-------------|
| **Season** | Ending year (2025 = 2024-25 season) |
| **GameID** | Unique game identifier |
| **DateOfGame** | Game date (YYYY-MM-DD) |
| **Visitor/Home** | Team names |
| **VisitorRank/HomeRank** | KenPom rankings on game day |
| **VisitorPred/HomePred** | Predicted final scores |
| **HomeWP** | Home team win probability (%) |
| **PredTempo** | Predicted tempo (possessions) |
| **ThrillScore** | Expected excitement level |

### Understanding Win Probability (HomeWP)

- **45-55%**: Toss-up game
- **55-65%**: Slight favorite
- **65-75%**: Moderate favorite
- **75-85%**: Strong favorite
- **85%+**: Heavy favorite
