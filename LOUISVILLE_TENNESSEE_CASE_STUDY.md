### 3. Recent Form Weighting ðŸ”¨ **BUILD THIS**

**Current Problem**:
- KenPom uses full season efficiency (all games weighted equally)
- Tennessee's recent 3-game skid not weighted more heavily
- Short-term trends matter for betting

**Solution**:
- Calculate "Last 5 Games" efficiency metrics
- Weight recent games 2-3x more heavily
- Track momentum shifts

**Implementation**:
```python
def calculate_recent_form_adjustment(team, games=5):
    recent_efficiency = get_last_n_games_efficiency(team, n=games)
    season_efficiency = team.season_adj_em

    # If recent form differs significantly from season average
    if abs(recent_efficiency - season_efficiency) > 5:
        weight_recent = 0.3  # 30% weight to last 5 games
        adjusted = (weight_recent * recent_efficiency +
                   (1-weight_recent) * season_efficiency)
        return adjusted - season_efficiency
    return 0
```
